# Snake Game - Base Project (Unity)

Proyecto base para la construcción de un videojuego tipo Snake en Unity.

## Requisitos
- Unity Hub
- Versión de Unity: 6000.4.2f1

## Instrucciones
1. Descargar el proyecto (Code > Download ZIP)
2. Descomprimir
3. Abrir desde Unity Hub
4. Cargar la escena principal
5. Ejecutar el juego

## Controles
- Flechas / WASD → mover
- R → reiniciar (si aplica)

## Objetivo
Este proyecto servirá como base para que los equipos desarrollen nuevas funcionalidades y realicen pruebas automatizadas

## Notas
No modificar la estructura base del proyecto.
